學習歷程
第二階段準備指南

把手冊中的原則、檔案勾選、自述與綜整心得，濃縮成一套可以現場教學、學生自查的網頁投影片。

備審資料不是「放越多越好」，而是要讓教授快速看見你與學系的關聯。

第二階段準備的核心，是把三年經歷整理成有證據、有反思、有方向的申請敘事。

審查心法

三重二不

大學審查重視校系需求、學習脈絡與能力證據；不追求資料堆疊，也不以名目華麗取代真實學習。

重學系要求與能力特質
重課程成果與反思脈絡
重學生自主與真實證據
不堆量、不包裝過度

三大任務

TASK 01

勾選檔案

不是全放，而是選出最能證明能力與學系連結的材料。

TASK 02

自述撰寫

用故事說明三年學習如何指向現在的申請選擇。

TASK 03

綜整心得

把多元表現整理成清楚、可讀、能被審查的能力證據。

FILE SELECTION
TASK 01

勾選要有理由

每一份檔案都要回答：它證明了什麼能力？和學系要求有什麼關係？教授讀完會記得哪個重點？

  • 先看學系準備指引，再回頭挑作品。
  • 優先選有過程、有反思、有修正的成果。
  • 避免重複性太高、只展示活動名稱的材料。
TASK 02

自述是一條線

高中學習歷程反思不只是流水帳，而是把經歷、選擇、能力與未來方向串成一條清楚的線。

  • 以「我為何適合這個學系」作為中心問題。
  • 用具體事件說明能力，不只寫形容詞。
  • 讓過去經驗自然連到大學學習計畫。

INTEGRATION
TASK 03

綜整要可讀

多元表現綜整心得的價值，在於把分散經驗壓縮成清楚架構，讓審查者看見你的能力輪廓。

  • 先分類，再摘要，不把所有經歷平均陳列。
  • 用簡表與標題降低閱讀負擔。
  • 文字要短，但反思要準，避免空泛口號。

雷達圖

審查資料要看見的五種能力

決策樹

不知道該放哪份？

先問：這份資料能不能證明「我適合這個學系」？
YES

放入主線

能清楚支援申請動機與能力，放在自述或重點檔案中。

MAYBE

補上反思

作品不弱但連結不明，補充學到什麼、如何修正與下一步。

NO

降低權重

與學系關聯低或重複性高，可不選，或放在輔助位置。

SOIL Workflow

六步驟完成準備

1

讀指引

圈出學系要求、能力特質、指定問題與字數限制。

2

盤點資料

把課程成果、多元表現、獎項、專題與反思列成清單。

3

選主線

決定最能代表你的三到四個能力證據。

4

寫故事

用經歷、轉折、學習、未來方向串成自述。

5

做綜整

用分類、簡表與小標題提升閱讀效率。

6

練追問

針對每份材料準備教授可能問的原因、細節與反思。

最後檢核

讓資料替你說出
清楚、真實、有方向

每份檔案都能說明能力與學系關聯
自述不是流水帳，而有明確主線
綜整心得有分類、摘要與反思
面試可回到資料細節與真實經驗
完成後：用 90 秒講出你的申請故事
